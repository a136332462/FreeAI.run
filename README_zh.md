## 这些页面分别在做什么

- best AI tools

这里既是用户第一次进入站点的入口，也是后续进入分类、搜索、免费工具页和产品详情页的分发层。

- AI tool search

- find AI tools for a task

这类用户通常不想慢慢逛，他们要的是更快缩小选择范围。

分类页适合“知道自己要做什么，但还没决定选哪个产品”的流量，例如：

- AI tools by category

分类页的作用，是把宽泛需求拆成更容易比较的具体方向。

免费 AI 工具页适合价格敏感和试用优先的搜索，例如：
